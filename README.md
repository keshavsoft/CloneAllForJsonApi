# CloneAllForJsonApi
use git cli to clone all the repositories needed from command line using a single batch file

on windows systems:

1.  run the BoilerPlate.bat file.
2.  go to the JsonApi repo

from windows terminal
or
vscode terminal
run the below commands

1.  StartBatch
2.  PrepareBatch
3.  cd JSONApi 
4.  node app
