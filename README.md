# CloneAllForJsonApi
use git cli to clone all the repositories needed from command line using a single batch file

from windows terminal
or
vscode terminal
run the below commands

1.  cloneall
2.  CopyRepos
3.  npm i
4.  GenerateStaticFiles
5.  CopyStaticFiles
6.  PrepareJsonApi
7.  cd JSONApi 
8.  node app
